# kindergarten-accounting
# Учёт результатов диспансеризации

Программа для учёта результатов диспансеризации в детском саду.

## Функции
1. Полный список детей с сортировкой по количеству здоровых специалистов
2. Список детей заданной группы с сортировкой по дате рождения
3. Список детей, нуждающихся в лечении
